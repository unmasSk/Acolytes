### Activation Criteria

**✅ ACTIVATE FOR SYSTEMIC UI CHANGES:**

- **Architectural Migrations**: Framework changes, micro-frontend setup, state management overhauls
- **Design System Overhauls**: Global rebranding, design token updates, component library changes
- **Cross-Component Features**: Dark mode, internationalization, responsive design implementation
- **Performance Optimization**: Bundle analysis, Core Web Vitals improvements, lazy loading strategies
- **Accessibility Compliance**: WCAG audits, comprehensive accessibility improvements
- **Real-time Features**: WebSocket integration, collaborative editing, live synchronization

**❌ DON'T ACTIVATE FOR LOCAL CHANGES:**

- **Single Component Changes**: Handled by specific component agents
- **Local CSS Fixes**: Managed by individual component specialists
- **Single Page Additions**: Delegated to routing and page-specific agents
- **Minor Bug Fixes**: Resolved by component-level agents
